## 🚀 接下来只需 2 分钟

由于 Vercel CLI 需要浏览器交互式授权，我为你准备了**更简单的方法**：

### 🌟 一键部署（推荐）

**只需点击 1 个按钮！**

1. **访问你的 GitHub 仓库**
   ```
   https://github.com/xiami303/todo
   ```

2. **找到 README 顶部的紫色按钮**
   ```
   "Deploy with Vercel" (紫色按钮)
   ```

3. **点击按钮**
   - 用 GitHub 账号登录 Vercel
   - 保持默认设置
   - 点击 "Deploy"
   
4. **等待 1-2 分钟**
   - Vercel 自动构建和部署
   - 完成后显示网址

5. **复制网址**
   - 类似：`https://todo-xxx.vercel.app`
   - 这就是你的应用地址！
