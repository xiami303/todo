# 🎤 语音待办事项

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fxiami303%2Ftodo)

支持语音输入的智能待办清单应用

## ✨ 功能特性

- 🎤 **语音输入** - 支持中文语音识别，快速创建待办事项
- ✅ **任务管理** - 添加、编辑、删除和完成任务
- 🎯 **优先级** - 高、中、低三级优先级管理
- 📁 **分类管理** - 个人、工作、购物、健康、其他分类
- 🔍 **搜索筛选** - 多维度筛选和搜索
- 📊 **数据统计** - 实时显示任务完成情况
- 💾 **本地存储** - 数据保存在浏览器本地，无需登录
- 📱 **PWA 支持** - 可安装到手机主屏幕，离线使用

## 🚀 快速开始

### 在线使用（推荐）

1. 访问部署的网址
2. 在 iPhone 上使用 Safari 打开
3. 点击底部分享按钮 → 添加到主屏幕
4. 像原生应用一样使用

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 📱 在 iPhone 上安装

1. **访问网址** - 用 Safari 浏览器打开应用网址
2. **添加到主屏幕**
   - 点击底部中间的分享按钮 📤
   - 向下滚动，找到"添加到主屏幕" ➕
   - 点击"添加"
3. **开始使用** - 在主屏幕找到应用图标，点击打开

## 🎤 使用语音输入

1. 点击"🎤 点击开始语音输入"按钮
2. 允许麦克风权限
3. 清晰说出待办事项
4. 系统自动识别并填充
5. 选择优先级和分类
6. 点击添加完成

## 🛠️ 技术栈

- **前端框架**: React 18
- **类型系统**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **语音识别**: Web Speech API
- **部署**: Vercel
