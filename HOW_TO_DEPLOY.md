# 📱 立即部署到 iPhone - 完整操作指南

**总用时：5 分钟** ⏱️

---

## 🎯 部署步骤（3步完成）

### 第 1 步：访问 GitHub 仓库

1. 打开浏览器访问：**https://github.com/xiami303/todo**
2. 找到 README 中的部署按钮：
   
   [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/xiami303/todo)

3. **点击这个按钮**

---

### 第 2 步：在 Vercel 部署

点击按钮后会跳转到 Vercel 页面：

1. **登录 Vercel**
   - 选择 "Continue with GitHub"
   - 使用你的 GitHub 账号登录

2. **创建仓库**（首次部署）
   - Repository Name: `voice-todo-app`（可自定义）
   - 保持其他默认设置
   - 点击 **"Create"**

3. **开始部署**
   - Vercel 自动开始部署
   - 等待 1-2 分钟
   - 看到 "🎉 Congratulations!" 表示成功

4. **获取网址**
   - 部署成功后会显示网址
   - 类似：`https://voice-todo-app-xxx.vercel.app`
   - **复制这个网址**

---

### 第 3 步：在 iPhone 上安装

现在用 iPhone 访问刚才的网址：

1. **打开 iPhone 的 Safari 浏览器**
   - ⚠️ 必须用 Safari，其他浏览器不支持

2. **访问网址**
   - 输入或粘贴 Vercel 给你的网址
   - 如：`https://voice-todo-app-xxx.vercel.app`

3. **添加到主屏幕**
   - 点击底部中间的 **分享** 图标 📤
   - 向下滚动
   - 找到并点击 **"添加到主屏幕"** ➕

4. **确认添加**
   - 应用名称默认为 "语音Todo"
   - 可以修改为你喜欢的名称
   - 点击右上角 **"添加"**

5. **完成！** 🎉
   - 返回 iPhone 主屏幕
   - 看到新的应用图标
   - 点击打开

---

## 🎤 首次使用语音功能

打开应用后：

1. **点击 "🎤 点击开始语音输入" 按钮**

2. **允许麦克风权限**
   - 首次使用会弹出权限请求
   - 点击 **"允许"**

3. **开始说话**
   - 清晰地说出待办事项
   - 例如："明天下午三点开会"
   - 例如："买牛奶和面包"

4. **自动填充**
   - 说完后语音会自动转换为文字
   - 显示在输入框中

5. **添加任务**
   - 选择优先级（高/中/低）
   - 选择分类（个人/工作/购物等）
   - 点击 **"添加"** 按钮

## 🎯 使用技巧

### 语音输入最佳实践

1. **环境**：选择安静的地方
2. **距离**：手机距离嘴巴 15-30cm
3. **语速**：正常说话速度即可
4. **内容**：清晰表达任务内容

### 示例语音输入

试试这些示例：
- "明天上午十点开部门会议"
- "周五前完成项目报告"
- "买鸡蛋、牛奶和面包"
- "每天早上七点跑步"
- "下午三点给客户打电话"

### 其他功能

- 📝 **编辑**：鼠标悬停任务，点击编辑图标
- ✅ **完成**：点击任务前的圆圈
- 🗑️ **删除**：鼠标悬停，点击删除图标
- 🔍 **搜索**：使用顶部搜索框
- 🎨 **筛选**：按优先级、分类筛选

## 🔧 常见问题

### Q: 找不到"添加到主屏幕"选项？

**A**: 
- 确认使用的是 Safari 浏览器
- 确认不在无痕模式
- 确认网站是 HTTPS（Vercel 自动提供）
- 尝试刷新页面

### Q: 语音识别不工作？

**A**: 
- 检查是否允许了麦克风权限
- 设置 → Safari → 麦克风 → 允许
- 确认在安静环境
- 尝试重启应用

### Q: 应用图标显示不正确？

**A**: 
- 删除应用重新添加
- 清除 Safari 缓存后重试
- 确认网络连接正常

### Q: 数据会丢失吗？

**A**: 
- 数据保存在本地，不会丢失
- 除非清除浏览器数据
- 建议定期截图备份重要任务
